# 🚀 ربات سیگنال کریپتو تلگرام

ربات پیشرفته تلگرام برای شناسایی ارزهای دیجیتال با احتمال پامپ بالا

---

## ✨ قابلیت‌ها

- 🔍 **اسکن ۴۰+ ارز** به صورت همزمان
- 📊 **تحلیل تکنیکال کامل**: RSI، MACD، Bollinger Bands، EMA 20/50
- 📦 **آنالیز حجم غیرمعمول** (Volume Spike Detection)
- 🎯 **احتمال پامپ** به صورت درصد (0-100%)
- 🟢 **نقطه ورود (خرید)** بر اساس سطوح حمایت
- 🔴 **اهداف قیمتی** (Target 1: +10%, Target 2: +20%)
- ⛔ **استاپ لاس** پیشنهادی
- 🔗 **لینک مستقیم** به CoinGecko و TradingView
- 🔔 **هشدار خودکار** با حداقل درصد قابل تنظیم
- 📰 **اخبار بازار** در لحظه
- 😱 **شاخص ترس/طمع** بازار

---

## 🛠 نصب و راه‌اندازی

### پیش‌نیازها
- Python 3.10+
- اکانت تلگرام
- توکن ربات از [@BotFather](https://t.me/BotFather)

### مراحل نصب

**۱. کلون کردن پروژه:**
```bash
git clone <repo_url>
cd crypto_bot
```

**۲. نصب وابستگی‌ها:**
```bash
pip install -r requirements.txt
```

**۳. تنظیم توکن:**
```bash
cp .env.example .env
```
فایل `.env` را باز کرده و `BOT_TOKEN` را با توکن ربات خود جایگزین کنید:
```
BOT_TOKEN=1234567890:AAFxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

**۴. اجرا:**
```bash
python bot.py
```

## 📱 دستورات ربات

| دستور | توضیح |
|-------|-------|
| `/start` | شروع ربات و منوی اصلی |
| `/scan` | اسکن کامل بازار |
| `/top` | برترین ۱۰ سیگنال پامپ |
| `/coin BTC` | تحلیل کامل یک ارز |
| `/alert ETH 70` | هشدار وقتی ETH بالای ۷۰% |
| `/news` | اخبار بازار |
| `/status` | وضعیت کلی بازار |

## 📊 روش محاسبه احتمال پامپ

امتیاز پامپ بر اساس این فاکتورها محاسبه می‌شود:

| فاکتور | وزن | توضیح |
|--------|-----|-------|
| حجم معاملات | ۲۵% | افزایش ناگهانی حجم = سیگنال مهم |
| RSI | ۲۰% | RSI زیر ۳۰ = اشباع فروش |
| MACD | ۲۰% | کراس صعودی = سیگنال مثبت |
| Bollinger Bands | ۱۵% | قیمت زیر باند پایین = فرصت |
| روند قیمت | ۱۰% | افت پس از نزول بلندمدت |
| مومنتوم ۱ ساعته | ۱۰% | شروع حرکت صعودی |
